Curandeiro Avançado

Há três guias disponíveis para personalização de seus perfis de recuperação:
<ul>
<li> Spell Healer permite que você personalize quais feitiços lançar com base em várias variáveis disponíveis.</li> 
<li> O Item Healer permite que você personalize quais itens usar com base em várias variáveis disponíveis.</li>
<li> Condition Healer permite que você personalize quais feitiços usar com base nas condições do seu personagem</li>
</ul>


Alertas configuráveis

Cada regra de alerta pode ser definida para reproduzir som, pausar a funcionalidade do bot e forçar o logout na lista de caracteres se qualquer um dos eventos descritos na primeira coluna com o valor configurável na última coluna for verdadeiro. Além disso, você pode adicionar nomes de criaturas que não acionarão o alerta e alterar os limites de valores para o restante das regras de alerta.




Perfis de segmentação e lista de segmentação

O Targeting é uma ferramenta poderosa para automatizar a seleção de alvos e as ações do seu personagem.
 Com o Targeting ativado, você pode configurar perfis personalizados para diferentes situações de caça e priorizar ações com base em uma variedade de critérios.\n\n

Você pode ter vários perfis de targeting. Cada perfil pode ser ajustado ao seu estilo de jogo, definindo prioridades:
<ul>
<li> Health - prioridades monstros com o menor percentual de saúde </li>
<li> Proximity - prioridades monstros que são os mais próximos do seu personagem </li> 
<li> Danger - prioridade monstros que têm o maior valor de "perigo" definido na configuração do monstro correspondente </li>
<li> Stick - prioriza a meta atual até morrer </li> 
</ul>\n 

Além disso, existem várias configurações que você pode definir para cada monstro definido na Lista de Targeting:
<ul>
<li> Avoidance - evite estar na frente/diagonal de uma criatura alvo </li> 
<li> tance - alcançar a criatura alvo ou manter distância dela </li> 
<li> Attack Mode - definido entre o modo ofensivo, equilibrado e defensivo </li> 
<li> HP Range - ataque a criatura com base em sua porcentagem de saúde </li>
<li> Entre outras... </li> 
</ul>

Lista de Waypoints, Guias e Rótulos

O CaveBot opera usando waypoints ou pontos de referência no mapa. Os jogadores podem criar rotas específicas indicando caminhos para seus personagens, 
além de ações a serem realizadas em pontos específicos, como usar cordas ou coletar itens.\n\n
Waypoints podem ser adicionados com base em 10 ações diferentes:
<ul>
<li> Node  - você pode ajustar o intervalo de um Nó para o valor desejado e o bot irá automaticamente para o próximo waypoint se seu personagem entrar no intervalo determinado </li> 
<li> Stand - vai para a posição exata de um waypoint (que pode ser ajustado para ir em qualquer uma das 8 direções) </li> 
<li> Action - usa o script fornecido na coluna Ação na Lista de Pontos de Passagem </li> 
<li> Shovel, Rope, Machete, Pick - usa o item com base no ID fornecido na caixa de texto correspondente nas configurações na posição do waypoint </li> 
<li> Entre outras... </li> 
</ul>
Além disso, você pode ter até 8 guias diferentes com waypoints separados e mover-se entre eles com base nas Ações que você fornece.


Scripts persistentes e teclas de atalho

Os Scripts Persistentes são executados periodicamente a cada período de tempo especificado até serem desabilitados.
Os scripts de teclas de atalho só são executados se a tecla escolhida antes for pressionada.
Você pode adicionar, remover e editar scripts rapidamente, bem como salvá-los e carregá-los do seu disco.
Cada script usado será incorporado ao arquivo salvo e carregado a partir dele para facilitar o compartilhamento de configurações de bot entre os usuários. \n

Lista de destinos e itens de saque

Cada Item de Saque que você deseja ser saqueado pode ser colocado em um Destino de Saque separado para mover automaticamente os itens para os contêineres configurados.
Várias opções, como Looting Range e Looting Policy, podem ser alteradas para melhor se adequarem às habilidades de jogo do seu personagem.
Além disso, ao selecionar as opções correspondentes, o bot pode abrir a próxima mochila se a anterior estiver cheia, abrir todas as mochilas no login e comer comida de cadáveres abertos.
